Exercise 1a ii:

Given the covariance matrix \(\Sigma\) of a sample \(x_1,x_2,x_3\in\mathbb{R}^3\)

\[ \Sigma = \begin{pmatrix} 2 & 0 & 1\\ 0 & 2 & 0\\ 0 & 0 & 3 \end{pmatrix}, \] calculate the principal components of this sample.

Solution

To calculate principal components, we first need to find the eigenvalues of \(\Sigma\). The eigenvalues \(\lambda\) are the solutions to the characteristic equation \(\text{det}(A - \lambda I) = 0\), where \(I\) is the identity matrix, and \(\text{det}\) denotes the determinant.

For the given matrix \(\Sigma\), the characteristic equation is given by: \[ \text{det}(\Sigma - \lambda I) = \begin{vmatrix} 2 - \lambda & 0 & 1\\ 0 & 2 - \lambda & 0\\ 0 & 0 & 3 - \lambda \end{vmatrix} = (2 - \lambda)(2 - \lambda)(3 - \lambda) \]

Setting this equation to zero and solving for \(\lambda\), we find three eigenvalues:

\[\begin{align*} \lambda_1 &= 2 \\ \lambda_2 &= 2 \\ \lambda_3 &= 3 \end{align*}\]

Next, we need to find the eigenvectors for each eigenvalue by solving the system of linear equations \((\Sigma - \lambda I) \nu = 0\), where \(\nu\) is the eigenvector corresponding to eigenvalue \(\lambda\).

For \(\lambda = 2\):

Substitute \(\lambda = 2\) into \((\Sigma - \lambda I)\nu = 0\) and solve for \(\mathbf{v}\):

\[ (\Sigma - 2I)\mathbf{v} = \begin{pmatrix} 0 & 0 & 1\\ 0 & 0 & 0\\ 0 & 0 & 1 \end{pmatrix} \mathbf{v} = 0 \]

The general solution is:

\[ \nu = \begin{pmatrix} x_1\\ x_2\\ 0 \end{pmatrix} \]

where \(x_1\) and \(x_2\) are arbitrary real constants.

For \(\lambda = 3\):

Substitute \(\lambda = 3\) into \((\Sigma - \lambda I)\nu=0\) and solve for \(\nu\): \[ (\Sigma - 3I)\nu = \begin{pmatrix} -1 & 0 & 1\\ 0 & -1 & 0\\ 0 & 0 & 0 \end{pmatrix} \mathbf{v} = 0 \]

The general solution is:

\[ \nu = \begin{pmatrix} x_1\\ 0\\ x_3 \end{pmatrix} \]

where \(x_1\) and \(x_3\) are arbitrary real constants.

Thus, the eigenvectors for the matrix \(\Sigma\) are:

\[ \nu_1 =\begin{pmatrix} 1\\ 0\\ 0 \end{pmatrix}, \quad \nu_2 = \begin{pmatrix} 0\\ 1\\ 0 \end{pmatrix} \quad \nu_3 = = \begin{pmatrix} 0\\ 0\\ 1 \end{pmatrix} \] Note, that the Eigenvectors have to be orthonormal! Those also correspond to the Principal components of the sample that generated \(\Sigma\).

Exercise 1b: Autoencoders

In your own words, explain the idea behind autoencoders.

Solution

The goal of autoencoders as introduced in lecture is similar to PCA: Dimension reduction. However, for autoencoders we train a neural network where the output is aimed to resemble the input. In the encoding phase, the data is compressed by passing it to a layer of size smaller than the input. This smaller layer represents the compressed data which we can extract and use for further experiments. In the decoding phase, the compressed data is passed to a final layer with the same size as the input. Here, the labels correspond to the input data, i.e., we try to reconstruct the input.

Exercise 2a ii:

Describe in your own words how we can proceed to calculate different distances of ordinal features.

Solution

Recall, that ordinal features can be grouped into descriptive categories that are ordered on some kind of hierarchical scale. Thus can apply the Gower coefficient instead:

\[ d_G(x_m,x_l) =\frac{\Sigma_{j=1}^J \delta(x_{mj},x_{lj})d(x_{mj},x_{lj})}{\Sigma_{j=1}^J \delta(x_{mj},x_{lj})}, \] where \[ d(x_{mj},x_{lj}) = \begin{cases}1,\quad x_{mj}\neq x_{lj},\\ 0, \quad x_{mj}\neq x_{lj}\end{cases} \] if the \(j\)-th variable is nominal/ordinal and

\[ d(x_{mj},x_{lj}) = \frac{\|x_{mj}-x_{lj}\|}{\max_{m}x_{mj}-\min_m x_{mj}} \] if the j-th variable is continuous.

Exercise 2c: K-Means

Similar to the hierarchical clustering algorithm, we can also deploy the k-means algorithm using the {tidyclust} package. However, before digging into the details of the k-means implementation we will first review some theoretical aspects.

Exercise 2c i:

Heuristically, describe the iterative nature of the k-means algorithm.

Solution

The iterative nature of the k-means algorithm finds its origin in the calculation of the means in each step: Initially, we set a predetermined number of centers randomly in the dataset and assign points to each center (cluster) for which the distance is minimal. Once every point has been assigned we calculate the mean of each cluster and set it as the new center. This iterative process is repeated until the algorithm has either converged (no new cluster assignments after a step), or a predetermined number of steps has been performed.
